# Jupyter Notebook

Canonical repo skill:

- `professor-x/skills/runtime/jupyter-notebook/SKILL.md`

Read the canonical repo skill before creating a notebook.

Minimum rules:

1. Define hypothesis, baseline, variables, and metrics first.
2. Make the notebook runnable top-to-bottom.
3. Keep cells small and single-purpose.
4. Record result summaries near the computation.
5. End with an explicit verdict and next experiment.
