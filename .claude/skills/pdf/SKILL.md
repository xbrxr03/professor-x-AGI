# PDF

Canonical repo skill:

- `professor-x/skills/runtime/pdf/SKILL.md`

Read the canonical repo skill before doing PDF-first research work. The point is
page-grounded evidence, not blind text extraction.

Minimum rules:

1. Record bibliographic metadata first.
2. Use extraction for search, not as the sole source of truth.
3. Ground strong claims to page or section.
4. Inspect figures and tables directly when they carry the result.
5. Separate quote, paraphrase, and inference.
