Use that workflow, not broad ideation:

1. Frame one hard research question.
2. Read local project context first.
3. Build the cross-disciplinary lattice.
4. Kill weak novelty claims quickly.
5. Keep only a surviving mechanism.
6. Define explicit kill criteria before promotion.
