## PHASE 1 — Make a weak local model EDIT reliably (the p_correct gap)  [NEXT, awaiting go]
Goal: an 8B local model in our harness lands correct edits without thrashing — the single biggest
"feels frontier" lever for weak models.
- **Edit-lever:** fuzzy search/replace, validate-before-apply (py_compile/parse), auto-retry on a failed
  apply, clear error feedback to the model.
- **Anti-thrash:** duplicate-action + loop detection, step budget, escalate-or-bail policy.
- **Behavior-keyed retrieval** (validated invention #1): on a failing task, inject the nearest past fix
  by failure-signature as a hint.
Goals/Milestones:
- M1.1: edit-apply success rate (applied-without-error) up measurably on the hard set.
- M1.2: duplicate-action / loop count down ≥50% vs current native baseline.
- M1.3 GATE: native repo-fix pass@1 (K-pass mean) ≥ pinned baseline 0.417 + MDE, on hard set +
  reuse-families. Report before/after honestly; if neutral, say so.

## PHASE 2 — Match frontier harness FEATURES for local models  [after P1]
Goal: table-stakes parity on the native (default) path.
- Port full scaffolding (skills/repair/repo/ICE) INTO the native tool-calling path (close the "native
  parity" gap where native scored lower only because it sent a stripped prompt).
- Land harness-gaps features on the main local path: MCP client, sub-agents/mirror critic, repo.map, ToT.
Milestones:
- M2.1 GATE: native path ≥ text path on repo-fix (K-pass), so native can be the sole default.
- M2.2: ablation shows each feature (repo.map, critic, ToT) is net-positive or it's cut.

## PHASE 3 — Self-improvement loop (the moat), verifier-gated  [after P2]
Goal: the system improves itself on usage — both levers — with a trust gate that prevents gaming.
- Lever 3 (harness/SkillOpt) + Lever 1 (distillation flywheel, un-paused on the clean GGUF; add the
  NaN/stop-sanity guard to the quant step).
- TGC accept-gate: commit an update only if it improves verifier pass@1 on RENAMED held-out anchors;
  log the train-vs-anchor Goodhart gap.
Milestones:
- M3.1 GATE: a self-improvement CURVE — held-out (renamed-anchor) pass@1 rises over N rounds.
- M3.2: Goodhart gap stays bounded (vs a train-gated baseline where it grows) → trust demonstrated.
