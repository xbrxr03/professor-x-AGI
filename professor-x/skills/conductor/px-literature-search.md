# px-literature-search

## Purpose
Gather and triage research evidence relevant to self-evolving local agent harnesses.

## Inputs
- `brain/questions.md`
- `brain/knowledge-base.md`
- `brain/hypotheses.md`

## Workflow
1. Start from open questions or hypotheses, not broad search.
2. Prefer local references and existing notes before network search.
3. If network is unavailable or disallowed, write a local-only literature backlog instead.
4. For every accepted claim, capture source id, claim, relevance, and uncertainty.
5. Do not update paper claims without a citation or local experiment id.

## Output Contract
Append validated findings to `brain/knowledge-base.md` or create a dated backlog in `professor-x/ops/daily/`.
