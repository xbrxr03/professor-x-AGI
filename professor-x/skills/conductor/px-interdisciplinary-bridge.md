# px-interdisciplinary-bridge

## Purpose
Find unnoticed patterns and new directions by importing structure from other
fields. Distilled from `deep-research/references/interdisciplinary_bridges.md`.
Most breakthroughs are a mature idea from field A applied to field B.

## The six bridge patterns (use as a checklist on any problem)
1. **Shared concept, different names** — the same mechanism exists elsewhere
   under another name (feedback loops = formative assessment = market correction
   = predictive coding). Find what your concept is called in 3 other fields.
2. **Shared method, different applications** — a method used elsewhere for a
   different purpose (network analysis: social ↔ citation ↔ neural).
3. **Complementary perspectives** — list how 5 disciplines would each frame the
   SAME phenomenon; the gaps between framings are where novelty hides.
4. **Theory migration** — a theory from field A adapted to field B (Complex
   Adaptive Systems: biology → management; Free Energy Principle: physics →
   neuroscience → ?).
5. **Methodological transfer** — borrow a mature method wholesale, adapting for
   the new field's constraints (RCT: medicine → education; A/B test: web →
   policy). Understand WHY it was designed that way before borrowing.
6. **Problem reframing** — redefine the same problem from each discipline's
   view; each yields different questions, methods, solutions. The integration of
   several framings usually beats any single one.

## Method
1. Define the core problem in ONE sentence (forces focus).
2. List 3 unfamiliar fields that face a structurally-similar problem.
3. For each: how would they frame it? What concept/method would they bring?
4. Borrow ONE concept/method; translate it into this project's language;
   state what you adapted and why.

## Pitfalls
- Surface borrowing (jargon without the underlying theory).
- Treating a field as monolithic ("economists believe…" — which school?).
- Forcing a method onto the wrong kind of question (measure vs understand).

## Output Contract
A short bridge note: core problem (1 sentence), 3 source fields, the borrowed
structure, the adaptation, and the concrete new direction it suggests.
