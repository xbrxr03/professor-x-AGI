# Discipline Lattice

Use this checklist to build the minimum cross-disciplinary map for invention
work on cognition, intelligence, or consciousness-like systems.

## Core fields

- neuroscience
- psychology
- philosophy of mind
- computer science
- machine learning
- control theory
- information theory
- mathematics of dynamics and geometry

## High-value adjacent fields

- quant finance
- mechanism design
- cybernetics
- complex systems
- thermodynamics / nonequilibrium physics
- robotics / embodiment
- causal inference
- computational phenomenology

## Questions to ask per field

- What is the field's core object?
- What does it measure well?
- What invariants does it care about?
- What failure modes does it already understand?
- What mature method could transfer?
- What language collision might hide overlap with another field?

## Transfer targets

Common things worth borrowing across fields:

- invariants
- state variables
- conserved quantities
- pricing / clearing rules
- path dependence
- regime detection
- hierarchy across timescales
- intervention logic
